# Game-of-Life
Conway's Game of Life. 

## Usage

Clone the repository. 

```javascript
git clone https://github.com/thomas1242/Game-of-Life/
```

Compile and run the source code.

```javascript
javac Game-of-Life/GameOfLife.java
java -cp Game-of-Life GameOfLife
```
